## Development

Using `pnpm`

1. Install dependencies

```sh
pnpm install
```

2. Run the project in development mode

```
pnpm dev
```

3. Lint the project

```
pnpm lint
```

## Run tests

Run unit tests once

```sh
pnpm test:unit
```

Run unit tests in watch mode

```sh
pnpm test:unit -- --watch
```

## Build for production

Build production mode

```
pnpm build
```
